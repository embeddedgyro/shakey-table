Detailed Description

Implementation of the MPU6050Interface. This is where the magic happens for calculating the cup holder's angular position based on the IMU measurements. The caclulated angular position is then sent as input to the outer PID controller as the process variable.

Implementation of the MPU6050Interface. This is where the magic happens for calculating the cup holder's angular position based on the IMU measurements. The caclulated angular position is then sent as input to the outer PID controller as the PV.
